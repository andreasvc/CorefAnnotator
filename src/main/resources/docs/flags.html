Flags

Since version 1.8, it is possible to add custom flags that can be attached to mentions, entities and detached mention parts. Flags are stored in the document.

Flag Editor

The flag editor can be opened via the tools menu. The editor shows a table with four columns, each row representing one flag. A flag is defined by four features (shown in the table from left to right):

  1. Icon: Each flag is associated with an icon to help annotators to recognize it. The icon can be selected from the selection shown on this page: aalmiray.github.io/ikonli/cheat-sheet-materialdesign.html.
